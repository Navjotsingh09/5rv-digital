## Active Scripts — Webflow Dir

Run these from inside `5rv-revamp.webflow/` unless noted.

| Script | Purpose | Run when |
|---|---|---|
| `_build_rv_components.py` | Regenerates `css/rv-components.css` from the canonical style definitions embedded in the script | Brand colour / button style changes |
| `_patch_pages.py` | Injects `<link href="css/rv-components.css">` into every HTML page and removes inline `rv-*` style blocks | After fresh Webflow export or CSS link is missing |
| `_patch_aria.py` | Adds `aria-controls`, `aria-haspopup`, `id="rv-mega-drop"` to all pages | After fresh Webflow export replaces nav HTML |
| `_unify_faq_v3.py` | Converts FAQ accordion triggers to native `<button>` with `aria-expanded` | After fresh FAQ section import from Webflow |
| `add_preloads.py` | Injects font `<link rel="preload">` + async CSS loading pattern | After significant CSS additions |
| `fix_blocking.py` | Removes render-blocking `<link>` tags from `<head>` | Performance audit failures |
| `fix_cls_bp.py` | Fixes CLS layout shifts and Best Practices Lighthouse issues | After image / layout changes |
| `fix_height_auto.py` | Removes invalid `height="Auto"` attributes on HTML elements | After Webflow re-export |
| `_inject_mega_css.py` | Re-injects mega menu CSS + hover bridge into all pages | If mega menu styling breaks |
| `_inject_rv2_bridge.py` | Adds hover bridge for top-nav Services panel | If nav dropdown closes prematurely |
| `_replace_clear_steps.py` | Replaces redundant section on `services.html` | One-off — already applied |
| `_replace_parallax_cta.py` | Replaces 4 parallax CTAs on `services.html` | One-off — already applied |
| `generate_case_studies.py` | Generates case study HTML pages from a CSV/data source | New case studies |
| `write_revamp.py` | Generates the revamped homepage `index.html` | Homepage content overhaul |

## Standard Deployment Workflow

```bash
# 1. Export HTML from Webflow → overwrite the .webflow/ folder contents

# 2. From 5rv-revamp.webflow/:
python3 _build_rv_components.py   # rebuild CSS
python3 _patch_pages.py           # inject CSS links
python3 _patch_aria.py            # ARIA fixes

# 3. Verify
grep -l 'rv-components.css' *.html | wc -l   # should equal total page count

# 4. Commit & deploy
git add -A
git commit -m "chore: post-export patch $(date +%Y-%m-%d)"
git push origin main    # triggers Vercel production deploy
```

## Branching Strategy

| Branch | Purpose |
|---|---|
| `main` | Production — every push auto-deploys to https://5rv.digital |
| `staging` | Pre-production preview — Vercel creates a unique preview URL |

Always branch from `main` for new features; merge to `staging` for review; merge to `main` for deployment.

## CSS Architecture

| File | Role |
|---|---|
| `normalize.css` | Browser reset |
| `webflow.css` | Webflow base styles (do not edit) |
| `5rv-revamp.webflow.css` | Main Webflow design (do not edit; re-exported from Webflow) |
| `css/rv-components.css` | **Custom component overrides** — edit via `_build_rv_components.py` |

**Brand colours:** Primary blue `#336bf9` (WCAG AA ≥ 4.5:1 on white). Do **not** revert to `#356eff` (contrast 4.35:1, fails AA for normal text).
