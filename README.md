# 5RV Digital - Tablet & Mobile Responsive Website

## Project Overview
5RV Digital is a full-featured responsive digital marketing agency website built with HTML, CSS, and JavaScript in Webflow.

## 📱 Responsive Breakpoints

### Mobile First Approach
- **Extra Small (0-479px)**: Phones (iPhone SE, Android phones)
- **Small (480-767px)**: Larger phones & small tablets
- **Tablet Portrait (768-859px)**: iPad Portrait mode
- **Tablet Landscape (860-1024px)**: iPad Landscape & larger tablets
- **Desktop (1025-1279px)**: Laptops & desktops
- **Large Desktop (1280-1439px)**: Larger desktops
- **Extra Large (1440-1919px)**: Large displays
- **Ultra Wide (1920px+)**: Ultra-wide screens

### Tablet Support
✅ **iPad**: All sizes (7", 8", 10", 12", 13")
✅ **Android Tablets**: All standard tablet sizes
✅ **Portrait & Landscape modes**: Optimized layouts for both orientations
